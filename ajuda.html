Touch Screen

Touch screen, ou tela sensível ao toque, é uma interface de entrada que permite aos usuários interagirem diretamente com o que é exibido no dispositivo, tocando a tela. Isso elimina a necessidade de outros periféricos, como teclado ou mouse. Use-a para arrastar as imagens para o lado para ver novas fotos.

Sem Touch Screen

Use o clique do mouse para interagir na imagem e use as setas do teclado para navegar pelas imagens.
